# Feature Spec: M56 Memory Provider Card Selection

## Goal

Bring Settings > Memory provider selection closer to the Arkloop-style card selection pattern while keeping Loomi copy and safety boundaries.

## Requirements

- Replace the compact segmented provider selector with card-like provider choices.
- Show Local, Nowledge, and OpenViking as selectable options.
- Keep enablement toggles, status, diagnostics, and Configure behavior unchanged.
- Preserve responsive layout on small screens.

## Non-Goals

- Do not add provider install/restart/process control.
- Do not change backend provider persistence.
- Do not copy external branding beyond provider names.
