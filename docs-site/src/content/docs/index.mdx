## 当前项目原则

Loomi 的目标是学习和复刻 Agent 平台的核心机制，但坚持机制对标、表达自研。项目优先构建可运行、可测试、可解释的薄片；先完成核心链路，再逐步扩展复杂桌面运行时、插件、Channels、Activity Recorder 和多 Agent 能力。

开发时默认要求同步更新本文档站。除非只是修拼写、格式化或无行为变化的小改动，否则代码、规格、架构说明、开发日志和验证结果应该一起更新。
