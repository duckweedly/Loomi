## 关联阶段

本次变更支撑 Loomi 的工程治理和长期知识沉淀，不改变现有 Web/Electron 产品行为。

## 完成内容

创建 `docs-site/` 作为独立 Starlight 文档站，用于集中展示路线图、架构说明、Spec Kit 工作流、开发日志和技术决策。

建立首页、阶段路线图、架构总览、Spec Kit 工作流说明和开发同步文档规则。后续非平凡开发应同步更新文档站，不需要用户单独提醒。

## 验证结果

已执行 `bun run build`，文档站构建通过。构建产物包含 13 个页面，并由 Starlight/Pagefind 生成搜索索引。

已有 `web/` 构建在接入 Spec Kit 后通过，只有 Vite chunk 体积警告。

## 已知限制

当前文档站是首版骨架，尚未把 `docs/` 下所有历史材料逐篇迁移为 Starlight 页面，也尚未建立 API、数据模型和 runbook 专区内容。

## 下一步

建议第一个通过 Spec Kit 管理的功能是 M2 API 与数据库基座。实现时应同步补充 API/DB 架构页、开发日志和必要 ADR。
